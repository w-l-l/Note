# 基本类型

## 类型声明

类型声明是 `TS` 非常重要的特点，通过类型声明可以指定 `TS` 中变量（参数，形参）的类型。

指定类型后，当为变量赋值时，`TS` 编译器会自动检查值是否符合类型声明，符合则赋值，否则报错。

简言之，类型声明给变量设置了类型，使得变量只能存储某种类型的值。

```ts
// let 变量: 类型
let x: string

// let 变量: 类型 = 值
let x: string = '1'

// function 函数名(参数: 类型): 返回值类型 {...}
function foo(a: number, b: number): number {
  return a + b
}
```
